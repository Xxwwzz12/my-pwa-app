Тестирование API Уведомлений

Нажмите кнопку для запуска теста

Инструкция по тестированию:

  1. Откройте главную страницу приложения в этой же вкладке
  2. Войдите в систему через Google OAuth
  3. Вернитесь на эту страницу и нажмите "Запустить тест"
  4. Если ошибка сохраняется, проверьте консоль разработчика (F12)
